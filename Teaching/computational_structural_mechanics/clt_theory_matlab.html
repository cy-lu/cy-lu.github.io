1. Basic definitions: ply, lamina, laminate

The most basic structural unit is the ply. In composite mechanics, a ply is one thin layer of fiber-reinforced material with a specific fiber direction. When that ply is treated as a mechanically homogeneous orthotropic layer, it is also called a lamina.

  • Ply / lamina: one single layer with one material orientation
  • Laminate: a stack of plies bonded together
  • Stacking sequence: the ordered list of ply angles, for example \([0/45/-45/90]_s\)
  • Ply angle \(\theta_k\): the orientation of the \(k\)-th ply relative to the laminate axes
  • Ply thickness \(t_k\): the thickness of the \(k\)-th ply

In a laminate, each ply may have the same material system but a different angle \(\theta_k\). This is what gives composite laminates their directional stiffness and design flexibility.

2. Why CLT is useful

Classical Lamination Theory is used because it gives the global mechanical response of a laminate quickly. Before any detailed finite element model is built, CLT can tell us:

  • the membrane stiffness of the laminate,
  • the bending stiffness of the laminate,
  • whether membrane–bending coupling is present,
  • the mid-plane strain under a given load,
  • the curvature under a given load,
  • the stress state in each ply.

In other words, CLT is the right first model when the objective is to understand how stacking sequence, ply thickness, and material orientation affect the structural response.

3. Material parameters and where they come from

A lamina needs both elastic constants and strength parameters.

Elastic constants

  • \(E_1\): longitudinal Young’s modulus
  • \(E_2\): transverse Young’s modulus
  • \(G_{12}\): in-plane shear modulus
  • \(\nu_{12}\): major Poisson’s ratio

Strength parameters

  • \(X_t\): longitudinal tensile strength
  • \(X_c\): longitudinal compressive strength
  • \(Y_t\): transverse tensile strength
  • \(Y_c\): transverse compressive strength
  • \(S\): in-plane shear strength

In practice, these values usually come from three places:

  1. manufacturer datasheets,
  2. published lamina data from books or papers,
  3. experimental identification from coupon tests.

Important distinction. Elastic constants determine deformation and stress distribution. Strength parameters determine whether that stress state is acceptable.

If only constituent-level data are available, one can estimate some lamina properties using micromechanics. For example, the simplest estimate for the longitudinal modulus is

\[ E_1 \approx V_f E_f + (1-V_f)E_m, \]

where \(V_f\) is the fiber volume fraction, \(E_f\) is the fiber modulus, and \(E_m\) is the matrix modulus. But in engineering work, real lamina data from a defined material system is preferred.

4. Lamina constitutive law

In the lamina material axes \((1,2)\), the in-plane stress vector and strain vector are

\[ \boldsymbol{\sigma}_{12}= \begin{bmatrix} \sigma_1\\ \sigma_2\\ \tau_{12} \end{bmatrix}, \qquad \boldsymbol{\varepsilon}_{12}= \begin{bmatrix} \varepsilon_1\\ \varepsilon_2\\ \gamma_{12} \end{bmatrix}. \]

The constitutive law under plane stress is

\[ \boldsymbol{\sigma}_{12}=\mathbf{Q}\boldsymbol{\varepsilon}_{12}, \qquad \mathbf{Q}= \begin{bmatrix} Q_{11} & Q_{12} & 0\\ Q_{12} & Q_{22} & 0\\ 0 & 0 & Q_{66} \end{bmatrix}. \]

The reciprocal Poisson ratio is

\[ \nu_{21}=\nu_{12}\frac{E_2}{E_1}. \]

The reduced stiffness terms are

\[ Q_{11}=\frac{E_1}{1-\nu_{12}\nu_{21}},\qquad Q_{22}=\frac{E_2}{1-\nu_{12}\nu_{21}},\qquad Q_{12}=\frac{\nu_{12}E_2}{1-\nu_{12}\nu_{21}},\qquad Q_{66}=G_{12}. \]

5. Ply rotation and transformed stiffness

Each ply in a laminate is oriented at angle \(\theta_k\). Therefore the ply stiffness must be transformed from the material axes \((1,2)\) into the laminate axes \((x,y)\).

\[ \boldsymbol{\sigma}_{xy}= \bar{\mathbf{Q}}\boldsymbol{\varepsilon}_{xy}, \]

where

\[ \boldsymbol{\sigma}_{xy}= \begin{bmatrix} \sigma_x\\ \sigma_y\\ \tau_{xy} \end{bmatrix}, \qquad \boldsymbol{\varepsilon}_{xy}= \begin{bmatrix} \varepsilon_x\\ \varepsilon_y\\ \gamma_{xy} \end{bmatrix}. \]

Here \(\bar{\mathbf{Q}}\) is the transformed reduced stiffness matrix. This transformation is exactly why the stacking sequence matters.

6. Laminate matrices \(\mathbf{A},\mathbf{B},\mathbf{D}\)

Let \(z_0,z_1,\ldots,z_n\) denote the through-thickness interface coordinates measured from the laminate mid-plane. Then

\[ \mathbf{A}=\sum_{k=1}^{n}\bar{\mathbf{Q}}^{(k)}(z_k-z_{k-1}), \] \[ \mathbf{B}=\frac{1}{2}\sum_{k=1}^{n}\bar{\mathbf{Q}}^{(k)}(z_k^2-z_{k-1}^2), \] \[ \mathbf{D}=\frac{1}{3}\sum_{k=1}^{n}\bar{\mathbf{Q}}^{(k)}(z_k^3-z_{k-1}^3). \]
  • \(\mathbf{A}\): extensional stiffness
  • \(\mathbf{B}\): membrane–bending coupling stiffness
  • \(\mathbf{D}\): bending stiffness

If the laminate is symmetric about the mid-plane, then \(\mathbf{B}=\mathbf{0}\).

7. Resultants, mid-plane strain, and curvature

The laminate constitutive equation is

\[ \begin{bmatrix} \mathbf{N}\\ \mathbf{M} \end{bmatrix} = \begin{bmatrix} \mathbf{A} & \mathbf{B}\\ \mathbf{B} & \mathbf{D} \end{bmatrix} \begin{bmatrix} \boldsymbol{\varepsilon}_0\\ \boldsymbol{\kappa} \end{bmatrix}. \]

The load resultants are

\[ \mathbf{N}= \begin{bmatrix} N_x\\ N_y\\ N_{xy} \end{bmatrix}, \qquad \mathbf{M}= \begin{bmatrix} M_x\\ M_y\\ M_{xy} \end{bmatrix}. \]

The generalized deformation variables are

\[ \boldsymbol{\varepsilon}_0= \begin{bmatrix} \varepsilon_x^0\\ \varepsilon_y^0\\ \gamma_{xy}^0 \end{bmatrix}, \qquad \boldsymbol{\kappa}= \begin{bmatrix} \kappa_x\\ \kappa_y\\ \kappa_{xy} \end{bmatrix}. \]

8. Ply strain and ply stress recovery

Once \(\boldsymbol{\varepsilon}_0\) and \(\boldsymbol{\kappa}\) are known, the in-plane strain at thickness coordinate \(z\) is

\[ \boldsymbol{\varepsilon}_{xy}(z)=\boldsymbol{\varepsilon}_0 + z\boldsymbol{\kappa}. \]

The stress in the \(k\)-th ply is then

\[ \boldsymbol{\sigma}_{xy}^{(k)}(z)=\bar{\mathbf{Q}}^{(k)}\boldsymbol{\varepsilon}_{xy}(z). \]

If a failure criterion is defined in the material frame, then \(\boldsymbol{\sigma}_{xy}^{(k)}\) must be transformed back into \(\boldsymbol{\sigma}_{12}^{(k)}=[\sigma_1,\sigma_2,\tau_{12}]^T\).

9. Worked example: carbon/epoxy laminate

Use representative carbon/epoxy lamina data:

QuantityMeaningValue
\(E_1\)longitudinal modulus135 GPa
\(E_2\)transverse modulus10 GPa
\(G_{12}\)in-plane shear modulus5 GPa
\(\nu_{12}\)major Poisson ratio0.30

Use laminate

\[ [0/45/-45/90]_s \]

with ply thickness

\[ t_{ply}=0.125~\text{mm}. \]

Apply load

\[ \mathbf{N}= \begin{bmatrix} 1000\\ 0\\ 0 \end{bmatrix} \text{N/m}, \qquad \mathbf{M}= \begin{bmatrix} 0\\ 0\\ 0 \end{bmatrix}. \]

The practical questions are:

  1. What are \(\mathbf{A},\mathbf{B},\mathbf{D}\)?
  2. Is \(\mathbf{B}=0\) for this symmetric laminate?
  3. What are \(\boldsymbol{\varepsilon}_0\) and \(\boldsymbol{\kappa}\)?
  4. Which ply carries the largest \(\sigma_1,\sigma_2,\tau_{12}\)?

10. Strength parameters and failure index

Introduce strength parameters:

  • \(X_t\): longitudinal tensile strength
  • \(X_c\): longitudinal compressive strength
  • \(Y_t\): transverse tensile strength
  • \(Y_c\): transverse compressive strength
  • \(S\): in-plane shear strength

A simple first-ply failure indicator is the Tsai–Hill index

\[ FI= \left(\frac{\sigma_1}{X}\right)^2 -\frac{\sigma_1\sigma_2}{X^2} +\left(\frac{\sigma_2}{Y}\right)^2 +\left(\frac{\tau_{12}}{S}\right)^2. \]

Here \(X\) and \(Y\) are chosen from \((X_t,X_c)\) and \((Y_t,Y_c)\) according to whether \(\sigma_1\) and \(\sigma_2\) are tensile or compressive.

Interpretation. \(FI<1\) suggests the ply is below the failure threshold, \(FI=1\) corresponds to failure onset, and \(FI>1\) indicates predicted first-ply failure.

11. MATLAB implementation

11.1 Main script

E1  = 135e9;
E2  = 10e9;
G12 = 5e9;
v12 = 0.30;

Xt = 1500e6;
Xc = 1200e6;
Yt = 40e6;
Yc = 150e6;
S  = 70e6;

angles = [0 45 -45 90 90 -45 45 0];
tply   = 0.125e-3;
NM     = [1000;0;0;0;0;0];

[ABD, z, Qbar_all] = build_laminate_ABD(E1,E2,G12,v12,angles,tply);

ek    = ABD \\ NM;
eps0  = ek(1:3);
kappa = ek(4:6);

disp('eps0 =');  disp(eps0)
disp('kappa ='); disp(kappa)

for k = 1:length(angles)
    zmid = 0.5*(z(k)+z(k+1));
    eps_xy = eps0 + zmid*kappa;
    sigma_xy = Qbar_all(:,:,k) * eps_xy;
    sigma_12 = stress_xy_to_12(sigma_xy, angles(k));

    s1  = sigma_12(1);
    s2  = sigma_12(2);
    t12 = sigma_12(3);

    if s1 >= 0, X = Xt; else, X = Xc; end
    if s2 >= 0, Y = Yt; else, Y = Yc; end

    FI = (s1/X)^2 - (s1*s2)/(X^2) + (s2/Y)^2 + (t12/S)^2;

    fprintf('Ply %d, angle = %+g deg\\n', k, angles(k));
    fprintf('sigma_12 = [%.3e %.3e %.3e] Pa\\n', s1, s2, t12);
    fprintf('Tsai-Hill FI = %.4f\\n\\n', FI);
end

11.2 Function: build laminate ABD

function [ABD, z, Qbar_all] = build_laminate_ABD(E1,E2,G12,v12,angles,tply)

v21 = v12 * E2 / E1;
den = 1 - v12*v21;

Q11 = E1 / den;
Q22 = E2 / den;
Q12 = v12 * E2 / den;
Q66 = G12;

Q = [Q11 Q12 0;
     Q12 Q22 0;
     0   0   Q66];

nply = length(angles);
h = nply * tply;
z = linspace(-h/2, h/2, nply+1);

A = zeros(3,3);
B = zeros(3,3);
D = zeros(3,3);
Qbar_all = zeros(3,3,nply);

for k = 1:nply
    theta = angles(k);
    m = cosd(theta);
    n = sind(theta);

    Qbar11 = Q11*m^4 + 2*(Q12+2*Q66)*m^2*n^2 + Q22*n^4;
    Qbar22 = Q11*n^4 + 2*(Q12+2*Q66)*m^2*n^2 + Q22*m^4;
    Qbar12 = (Q11+Q22-4*Q66)*m^2*n^2 + Q12*(m^4+n^4);
    Qbar16 = (Q11-Q12-2*Q66)*m^3*n - (Q22-Q12-2*Q66)*m*n^3;
    Qbar26 = (Q11-Q12-2*Q66)*m*n^3 - (Q22-Q12-2*Q66)*m^3*n;
    Qbar66 = (Q11+Q22-2*Q12-2*Q66)*m^2*n^2 + Q66*(m^4+n^4);

    Qbar = [Qbar11 Qbar12 Qbar16;
            Qbar12 Qbar22 Qbar26;
            Qbar16 Qbar26 Qbar66];

    Qbar_all(:,:,k) = Qbar;

    zk1 = z(k);
    zk2 = z(k+1);

    A = A + Qbar * (zk2 - zk1);
    B = B + 0.5 * Qbar * (zk2^2 - zk1^2);
    D = D + (1/3) * Qbar * (zk2^3 - zk1^3);
end

ABD = [A B;
       B D];
end

11.3 Function: transform stress back to material axes

function sigma_12 = stress_xy_to_12(sigma_xy, theta)

m = cosd(theta);
n = sind(theta);

sx  = sigma_xy(1);
sy  = sigma_xy(2);
txy = sigma_xy(3);

s1  = m^2*sx + n^2*sy + 2*m*n*txy;
s2  = n^2*sx + m^2*sy - 2*m*n*txy;
t12 = -m*n*sx + m*n*sy + (m^2-n^2)*txy;

sigma_12 = [s1; s2; t12];
end

The point of this code is not only to get numbers. It is to make the CLT workflow explicit and learnable.
