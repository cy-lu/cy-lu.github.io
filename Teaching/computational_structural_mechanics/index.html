Computational Structural Mechanics

Teaching Notes on Classical Lamination Theory, MATLAB, COMSOL, and the Bridge to Fracture Mechanics

This module is written as a learning path rather than a summary page. The aim is to start from the basic structural building block of a laminate, define the mechanical quantities carefully, derive the laminate stiffness, implement the workflow in MATLAB, and then connect the results to COMSOL-based validation and, later, to fracture mechanics.

The recommended reading order is: lamina and laminate definitions → CLT theory → MATLAB implementation → strength assessment → COMSOL validation → fracture bridge.

How to use these notes

  • If you are new to composite laminates, start with CLT Theory and MATLAB.
  • If you already know the equations and want to connect them to simulation, continue with COMSOL Validation and Fracture Bridge.
  • If you are using the notes for teaching, the first page is the core page and the second page is the engineering extension.
