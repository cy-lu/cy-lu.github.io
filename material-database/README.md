# FDTD-ready RP/CP Material Fit Database Website

This folder is generated by `build_material_fit_github_site_SPYDER_DIRECT.py`.

## Deployment

1. Commit the `docs/` folder to a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select branch `main` and folder `/docs`.

## What the website does

The website is a static browser for precomputed fitting results:

- material and record selection;
- N selection;
- RP vs CP-family decision;
- FDTD-readiness / G(theta) status;
- n, k, Re ε, Im ε plots;
- Tidy3D PoleResidue JSON download.

No fitting is performed in the browser.

## Core design choice

`CP family` means the better FDTD-ready model among:

- Analytical CP;
- SemiCP.

The user-facing website compares only:

- RP;
- CP family.

This keeps the interface simple while preserving internal traceability.
